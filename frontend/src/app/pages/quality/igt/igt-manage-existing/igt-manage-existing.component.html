Mark as Used

Serial {{ serialToMarkUsed?.serial_number }} will be marked as used and removed from available inventory.

When to use manual Mark as Used
  • Serial was physically used but not recorded through a serial assignment.
  • Offline or fallback production consumed this serial.
  • Internal sample, demo, or in-house testing.
Damaged or lost serials should use Write Off, not Mark as Used.
