Integration Guide
Basic Usage
<app-serial-number-generator 
  (serialNumberGenerated)="onSerialGenerated($event)">
</app-serial-number-generator>
With Template Configuration
<app-serial-number-generator 
  [config]="{ prefix: 'PROD', includeDate: true }"
  [autoGenerate]="true"
  (serialNumberGenerated)="onSerialGenerated($event)">
</app-serial-number-generator>
With Barcode Printing
<app-serial-number-generator 
  [config]="templateConfig"
  [showForm]="true"
  (serialNumberGenerated)="onSerialGenerated($event)">
</app-serial-number-generator>
Template System

The template system provides predefined configurations for common use cases:

  • Product Templates: Standard product serial numbers with date and random components
  • Asset Templates: Asset tracking tags optimized for equipment management
  • Operations Templates: Work orders, quality control, and service tickets
  • Financial Templates: Transaction IDs and audit trail numbers
  • Logistics Templates: Shipment tracking and batch numbers
